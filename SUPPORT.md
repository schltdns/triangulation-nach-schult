# Support

## 🇩🇪 Support‑Informationen
Vielen Dank für dein Interesse an DNS / frAIme.
Ich entwickle dieses Projekt als Einzelperson. Daher ist der Support bewusst klar strukturiert.

### Wie du Unterstützung erhältst
Bitte nutze die folgenden Wege:

**1. Issues**
Für Fragen, Fehlerberichte oder Verbesserungsvorschläge eröffne bitte ein Issue:
- klare Beschreibung
- reproduzierbare Schritte (falls relevant)
- Beispiel‑Prompt + Δdiv (falls relevant)

**2. Dokumentation**
Viele Fragen lassen sich durch die vorhandenen Markdown‑Dateien beantworten, insbesondere:
- P1–P8 Struktur
- Δdiv‑Berechnung
- Power‑Layer
- Beispiele und Case Studies

**3. Keine privaten Support‑Anfragen**
Ich kann keine individuellen Beratungen per E‑Mail oder Direktnachricht anbieten.
Bitte nutze Issues, damit alle von der Diskussion profitieren.

**4. Sicherheitsrelevante Themen**
Bitte keine Sicherheitsprobleme öffentlich posten.
Nutze stattdessen die Kontaktadresse in [SECURITY.md](SECURITY.md).

## 🇬🇧 Support Information
Thank you for your interest in DNS / frAIme.
I maintain this project as an individual, so support is intentionally structured and transparent.

### How to get support
Please use the following channels:

**1. Issues**
For questions, bug reports, or suggestions, open an Issue with:
- a clear description
- reproducible steps (if relevant)
- example prompt + Δdiv (if relevant)

**2. Documentation**
Many questions can be answered through the existing Markdown files, especially:
- P1–P8 structure
- Δdiv calculation
- Power layer
- Examples and case studies

**3. No private support**
I cannot provide individual consulting via email or direct messages.
Please use Issues so discussions remain public and helpful to others.

**4. Security‑related topics**
Please do not post security issues publicly.
Use the contact information in [SECURITY.md](SECURITY.md).
