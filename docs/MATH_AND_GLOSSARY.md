# DNS Glossary (v2.1)

**Δdiv**
`Δdiv = 0.5 × (1 - Jaccard) + 0.5 × (1 - Cosine)`

**Four Questions**
On topic, New idea, Verifiable, Understandable

**Divergence types**
- Substantive: factual disagreement
- Methodological: refusal / underspecification
- Normative: value conflict

**Falsification rules**
1. Persistent divergence → inconclusive
2. Refusal → back to P1
3. No justification → invalid
4. External error → back to P3

**P7 questions**
1. What dominated?
2. What ignored?
3. What unfalsified?
